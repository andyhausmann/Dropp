jQuery Dropdown Replacement

Introduction

Dropp is a jQuery plugin which replaces regular droprown menus ( <select> elements ) with stylable alternatives.

All you have to do is to initialize the replacer as shown below.

$('select').dropp();

It'll handle the rest!

 

Settings

You can specify settings by attaching them to initializer. See below for example:

$('select').dropp({ 'phrase_on_multiple' : 'Multiple Choices', 'class_dropdown_wrapper' : 'my_custom_dropdown_wrapper' });
 

Setting: "phrase_on_multiple"

Prints the specified text in multiple choice select elements when there are multiple choices selected.

Default is false

$('select').dropp({ 'phrase_on_multiple' : 'Multiple Choices' });

Setting: "class_dropdown_wrapper"

Sets the css class to be used in the dropdown wrapper element.

Default is dropdown_wrapper

$('select').dropp({ 'class_dropdown_wrapper' : 'my_custom_dropdown_wrapper' });
 

Setting: "class_dropdown_list"

Sets the css class to be used in the dropdown list wrapper element.

Default is dropdown_list

$('select').dropp({ 'class_dropdown_list' : 'my_custom_dropdown_list' });

Setting: "class_visible_dropdown"

Sets the css class to be used in the dropdowns clickable element.

Default is dropdown

$('select').dropp({ 'class_visible_dropdown' : 'my_custom_dropdown' });
 

Setting: "class_option_selected"

Sets the css class to be added to an item when it is selected.

Default is selected

$('select').dropp({ 'class_option_selected' : 'my_custom_dropdown_item_selected' });
